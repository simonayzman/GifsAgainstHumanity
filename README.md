GifsAgainstHumanity
===============

A simple setup to get started using Datadipity.com and NML.js for the GifsAgainstHumanity app at HackNY Spring 2015.


Dependencies
============
Node.js

npm

Leaflet.js

JQuery 2.x

NML.js v0.2.0

CryptoJS MD5

JSViews

Start App
=========
````
node server.js

http://localhost:5000
````
